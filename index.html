Polynomial Interpolation

You need a browser which supports canvas

How it works

If you have points [(x,y), ...] And say you want to find a polynomial of degree N
where N = len(points) How should you do it?

My immediate thought was to turn to linear algebra, as we can express our problems as linear systems of equations.
This might not be immediately obvious, but consider how a polynomial is only raising its x terms to the n'th power.
Since the x values of the points are known, we can transform this problem
Find a quadratic which intersects the points(1,2), (3,2), (2,6)
Into this (easier) problem
Solve the system ax^2 + bx + c = y for each (x,y) in points.
Note that since x is known, we can easily just evaluate them and get the system (in matrix form now) 

# each row is [1, x, x^2] for each x in points
# (backwards order makes evaluation in code easier)
A = [[1 1 1]
     [1 3 9]
     [1 2 4]]

b = [2, 2, 6] # what we want (y values for points)
x = [a, b, c] # unknowns, what we're solving for
Then it comes down to solving Ax = b. which is easy using gaussian elimination.
